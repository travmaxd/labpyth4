# Лабораторная работа №4
Графический интерфейс, параллельные вычисления и система тестирования
## Описание
Приложение демонстрирует работу различных генераторов в Python, включая:

-генератор абсолютных значений;

-генератор случайных паролей;

-генератор, перемножающий элементы двух списков;

-сравнение производительности последовательного и параллельного генераторов (бенчмарк).

В проект также входит полноценный графический интерфейс на PySide6 и система тестирования на pytest с покрытием кода.

---

## Инструкции по запуску

###  Установка зависимостей
Убедитесь, что установлен Python 3.9+, затем выполните:
```bash
pip install -r requirements.txt
```
## Используемые технологии

| Компонент | Назначение |
|------------|-------------|
| Python 3.9+ | Основной язык разработки |
| PySide6 | Создание графического интерфейса (GUI) |
| multiprocessing | Параллельная обработка |
| pytest + pytest-cov | тесты |
| Генераторы Python | Основная логика заданий |

---
###  Запуск графического интерфейса
```bash
python app.py
```

После запуска откроется окно с 4 вкладками:

Задание 1 — генерация абсолютных значений из диапазона;

Задание 2 — генерация паролей (последовательно и параллельно);

Задание 3 — перемножение элементов двух списков;

Бенчмарк — сравнение скорости генерации 50 000 паролей.



---

###  Тесты
```bash
python -m pytest tests -v
# или тесты с покрытием
python -m pytest -v --cov=.
```
